## Design Guidelines

### Text Overlays

**Typography**:
- **Headline Font**: SF Pro Display Bold, 48-60pt
- **Subtext Font**: SF Pro Text Regular, 28-36pt
- **Color**: White text with dark semi-transparent background or dark text on light background

**Positioning**:
- Place text overlays at top or bottom of screenshot
- Ensure text doesn't obscure important UI elements
- Use consistent positioning across all screenshots

**Contrast**:
- Ensure text is readable against background
- Use drop shadows or background overlays if needed
- Test readability at small sizes

### Visual Style

**Colors**:
- Use app's brand colors consistently
- Football: Green (#2D5016), Brown (#6B4423), White
- Accent: Gold (#FFD700) for achievements
- Background: Clean white or subtle gradients

**Consistency**:
- Use same device frame (or no frame) across all screenshots
- Maintain consistent UI elements (status bar, navigation)
- Use same user profile across screenshots

**Quality**:
- Export at 2x or 3x resolution for sharpness
- Avoid compression artifacts
- Ensure text is crisp and readable



## Screenshot Creation Process

### Option 1: Using Simulator + Design Tool

1. **Capture Screenshots**:
   - Run app in iOS Simulator at required sizes
   - Navigate to each screen
   - Capture screenshots (Cmd+S in Simulator)
   - Save as PNG files

2. **Add Overlays**:
   - Use Figma, Sketch, or Photoshop
   - Import screenshots
   - Add text overlays with proper typography
   - Add device frames (optional)
   - Export at required resolutions

3. **Optimize**:
   - Compress images without quality loss
   - Verify file sizes (max 8MB per screenshot)
   - Check that all text is readable

### Option 2: Using Screenshot Tools

**Recommended Tools**:
- **Figma**: Free, collaborative, great for overlays
- **Sketch**: Mac-only, professional design tool
- **Adobe Photoshop**: Full-featured image editing
- **Canva**: User-friendly, template-based
- **App Store Screenshot Generator**: Online tools like [appure.io](https://appure.io)

### Option 3: Using App Store Screenshot Services

**Professional Services**:
- **AppLaunchpad**: Automated screenshot generation
- **Shotbot**: AI-powered screenshot creation
- **PlaceIt**: Template-based mockups

## Notes

- **First 3 screenshots are most important**: They appear in search results
- **Show value quickly**: Users decide in seconds whether to download
- **Use real data**: Avoid lorem ipsum or obviously fake content
- **Test on device**: View screenshots on actual iPhone/iPad to check readability
- **Update regularly**: Refresh screenshots when adding major features
